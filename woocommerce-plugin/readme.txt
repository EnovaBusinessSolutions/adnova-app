Conecta tu tienda WooCommerce con ADRAY AI para auditorías de marketing automatizadas.

== Description ==

**ADRAY para WooCommerce** conecta tu tienda con la plataforma ADRAY AI para recibir auditorías de marketing automatizadas y análisis de rendimiento.

= Características =

* **Conexión Simple**: Conecta tu tienda en un clic
* **Webhooks Automáticos**: Sincronización automática de datos de pedidos, productos y clientes
* **Análisis con IA**: Recibe recomendaciones personalizadas basadas en inteligencia artificial
* **Panel Unificado**: Ve todos tus datos de marketing en un solo lugar

= Datos Sincronizados =

El plugin sincroniza automáticamente:

* Pedidos (creación, actualización, eliminación)
* Productos (creación, actualización, eliminación)
* Clientes (registro, actualización)
* Cupones (marketing y promociones)

= Requisitos =

* WordPress 5.8 o superior
* WooCommerce 5.0 o superior
* PHP 7.4 o superior
* Cuenta en ADRAY AI (https://adray.ai)

== Installation ==

1. Sube la carpeta `adray-woocommerce` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a WooCommerce → ADRAY en el panel de administración
4. Haz clic en "Conectar Tienda" para vincular con tu cuenta ADRAY

== Frequently Asked Questions ==

= ¿Necesito una cuenta en ADRAY? =

Sí, necesitas crear una cuenta gratuita en https://adray.ai para usar este plugin.

= ¿Qué datos se envían a ADRAY? =

El plugin envía datos de pedidos, productos, clientes y cupones para generar análisis de marketing. No se envían datos sensibles como contraseñas o información de pago.

= ¿Puedo desconectar mi tienda? =

Sí, puedes desconectar tu tienda en cualquier momento desde la página de configuración del plugin.
